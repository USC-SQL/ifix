Comment trouver un fichier que j’ai reçu dans Skype pour le bureau Windows ?

Cet article est également disponible pour :

Mac

Les fichiers que vous avez reçus de vos contacts se trouvent dans l’historique des conversations avec ce contact.

  1. Connectez-vous à Skype.
  2. Dans la liste des contacts, cliquez sur le contact qui vous a envoyé le fichier. L’historique de vos conversations avec ce contact est affiché.
  3. Recherchez le fichier reçu. 


     
  4. Cliquez sur Ouvrir le fichier ou Ouvrir le dossier de destination.

Pour obtenir la liste de tous les fichiers envoyés et reçus, cliquez sur Affichage > Fichiers envoyés et reçus  dans le menu principal. Cette liste contient le nom de l’expéditeur du fichier, le nom du fichier, ainsi que la date et l’heure de l’envoi. À l’aide de ces informations, vous pouvez rechercher le fichier dans l’historique des conversations avec votre contact. Vous ne pouvez pas ouvrir de fichier depuis la liste Fichiers envoyés et reçus.
